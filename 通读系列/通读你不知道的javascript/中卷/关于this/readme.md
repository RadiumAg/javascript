### 默认绑定

​	没有上下文的情况，直接调用函数，此时看调用栈.但是注意的是严格模式下是undefined.

### 隐式绑定

​	对象中的函数被调用,指Obj.foo() 这种形式.不包括对对象函数的引用之后调用,如let a = obj.foo;a();此时this丢失,应用的是默认绑定.

### 显式绑定

​	bind,call,apply函数

### new 绑定

​	使用new,对构造函数的调用

new 操作发生的事情:

1.创建（或者构造）一个全新的对象。

2.这个新对象会被执行[[原型]]链接。

3.这个新对象会被绑定到函数调用的this

4.如果函数没有返回其他对象，那么new表达式中的函数会自动返回这个新对象。
